## Studi Kasus
Sistem ini digunakan untuk mengkategorikan produk elektronik (Laptop, Smartphone, dan Tablet) berdasarkan harga.  
Program meminta input jenis produk dan harga, lalu menentukan kategori menggunakan struktur **Nesting If** di dalam **For Loop**.

- Input jenis produk (Laptop/Smartphone/Tablet)
- Input harga produk
- Menentukan kategori berdasarkan harga
- Menggunakan For Loop dan Nested If
